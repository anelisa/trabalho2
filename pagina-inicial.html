Trabalho final da disciplina de fundamentos do git

Este trabalho tem por objetivo colocar em pratica a utilização dos comandos do git de forma pratica.

Para isto foi proposto uma estrutura de trabalho em que será contido:

  • Iremos criar 4 branchs, duas para cada participante do grupo
  • Iremos testar comandos para atualizar nossa branchs, realizar merges, criar PRs, criar issues
  • Será feita também uma apresentação final e pratica demonstrando o que foi realizado neste projeto.

Trata-se de um projeto simples, porém que irá conter muita pratica em todo da utilização do git em cenario real.
